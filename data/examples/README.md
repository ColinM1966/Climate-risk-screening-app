# Example areas of interest

This folder contains small example spatial files used for testing the application.

Examples may include:

- Jambongan GeoPackage;
- a simple GeoJSON polygon;
- a KML example;
- a multipart polygon;
- a buffered point example.

Example files should:

- have a defined CRS;
- contain valid geometry where possible;
- be small enough for GitHub;
- include a clear ID or name field.
