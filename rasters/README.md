# Raster data

This folder contains the climate, coastal, and static raster datasets used by the Sabah Climate Risk Explorer.

Each raster should be stored only once.

Climate-dependent rasters are organised by scenario and time period:

```text
rasters/baseline/1981-2010/
rasters/future/ssp126/2011-2040/
rasters/future/ssp126/2041-2070/
rasters/future/ssp126/2071-2100/
rasters/future/ssp245/
rasters/future/ssp370/
rasters/future/ssp585/
